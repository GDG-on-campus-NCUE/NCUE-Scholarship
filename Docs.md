## 2. 角色權限控管

本平台依據使用者身分劃分三種權限等級，各層級擁有不同的功能存取權：

### 2.1 訪客 (Guest)

*   **瀏覽公告**：可查看所有公開的獎學金公告列表。
*   **篩選查詢**：依據「申請期限」、「關鍵字」篩選公告。
*   **下載資源**：可將公告內容匯出為 PDF 或下載附件檔案。

### 2.2 註冊學生 (Registered Student)

*   **包含所有訪客功能**。
*   **收藏公告**：可將感興趣的單篇公告內容，一鍵寄送至個人信箱備份。
*   **AI 智能助理**：無限制使用 AI 諮詢服務，協助整理以及釐清獎學金相關資訊。
*   **接收推播**：接收由管理員發送的重要公告 Email 通知。
*   **個人化服務**：管理個人帳號、修改學號/姓名。
*   **真人客服支援**：可透過系統直接將問題轉寄給承辦人員。

### 2.3 管理員 (Administrator)

*   **包含所有學生功能**。
*   **公告管理**：新增、編輯、下架、刪除公告，並可匯出 CSV 報表或 PDF 檔案。
*   **數據分析**：檢視公告瀏覽數趨勢圖、統計總瀏覽量與逾期公告。
*   **AI 輔助上架**：使用 AI 自動分析文件並生成公告草稿。
*   **精準推播**：可針對「特定身分」（如僅發給管理員或學生）或「特定個人」寄送通知信。
*   **系統設定**：管理系統參數與 API 配置。

## 3. 管理員操作指南

本章節專為管理員設計，詳細說明後台各項功能之操作與注意事項。

### 3.1 公告編輯與管理
![image](https://hackmd.io/_uploads/H1FDDCjLZg.png)

平台採用 **TinyMCE** 進階文本編輯器，支援 HTML 格式排版。

>[!Note] **重要限制與注意事項**
> * **使用額度**：目前使用免費版方案，每月上限為 **1000 次編輯載入** (約可新增/編輯 500 則公告)。
> * **額度超用**：若超過額度，編輯器將鎖定並顯示錯誤。此時需至 ==[Tiny Cloud](https://www.tiny.cloud/)== 申請新的 API Key，並於後台「系統設定」中更新金鑰方可恢復使用。
> * **狀態設定**：若希望公告暫存或不顯示於前台，請將狀態設為「下架/隱藏」。


### 3.2 數據統計與匯出
![image](https://hackmd.io/_uploads/Hy1fD0oUZl.png)

在「匯出與統計」頁籤中，提供完整的數據分析工具：

*   **瀏覽數趨勢圖**：可切換以「日」、「週」、「月」為單位，檢視平台流量變化。
*   **公告匯出 (CSV)**：
    1.  勾選欲匯出的公告（可全選）。
    2.  點擊「匯出」，系統將產生包含標題、摘要、瀏覽數等欄位的 CSV 報表。
    3.  **批次刪除**：亦可在此處批次勾選並刪除過期公告。
*   **單篇下載 (PDF)**：在公告列表中，點擊下載圖示即可將該篇公告轉為正式的 PDF 文件，便於列印歸檔。

### 3.3 AI 智能公告生成

系統提供 AI 輔助功能，可自動分析獎學金辦法並填寫欄位。

>[!Note] **AI 額度與金鑰配置**
> * **額度限制**：本平台採用的 Google Gemini API 有每日 / 每分鐘請求次數限制。
> * **額度超用**：若系統提示 AI 服務額度已達上限或無法回應，管理員可至 ==[Google AI Studio](https://aistudio.google.com/api-keys)== 申請新的 API Key 並綁定帳單，接著於後台「系統設定」中進行配置方可恢復使用權限。

**資料來源模式**：

1.  **檔案上傳 (推薦)**：直接上傳辦法檔案。
    *   ✅ **支援 AI 分析之格式**：PDF (文字型)、JPG、PNG。
    *   ❌ **不支援 AI 分析，但平台附檔支援之格式**：Word (.doc/docx)、Excel (.xls/xlsx)...。
2.  **網址連結**：提供外部網站連結。

    >[!Warning]**注意:**
    >AI 僅能讀取該網頁的「文字內容」，**無法**深入讀取該網頁內包含的附件檔案。若主要辦法在附件中，請下載後使用「檔案上傳」模式。

### 3.4 附件檔案管理

*   **數量限制**：單一公告最多支援 **8 個**附檔。
*   **大小限制**：單一檔案最大 **15 MB**。
*   **檔名規範 (Critical)**：使用者在下載時僅能透過檔名辨識內容，模糊的檔名會造成辨別困擾，因此上傳前**務必修改檔名**。
    *   ❌ 錯誤範例：`scan_20240101.pdf`、`attachment1.pdf`
    *   ✅ 正確範例：`113學年度申請辦法.pdf`、`xxx申請表.pdf`
*   **排序功能**：上傳後可拖曳以調整附檔顯示順序。

### 3.5 推播與通知系統

管理員擁有多種通知管道，請依需求選擇：

![image](https://hackmd.io/_uploads/rJZP_AsIWg.png)
1.  **Email 群發/個別通知**：
    *   **針對特定角色**：可選擇寄送給「所有使用者」、「僅一般學生」或「僅管理員」。
    *   **針對特定個人**：在使用者列表中找到該用戶，點擊「通知」圖示即可發送個別信件。
    *   **公告群發**：在公告發布後，可手動點擊「Email 推播」，將該則公告轉為 HTML 信件寄給所有註冊用戶。

2.  **LINE 公告推播 (半自動)**：
    *   **操作流程**：系統將公告轉為短訊格式 ⭢ 發送至管理員 LINE 官方帳號 ⭢ **由管理員手動轉發**至學生社群。
    *   **限制原因**：LINE 官方帳號免費額度有限，故採半自動模式運作。

### 3.6 學生提問與回覆

當學生透過 AI 助理點擊「尋求真人協助」時，系統會發送郵件至 ==act5718@gmail.com==。

**回覆標準流程**：

1.  開啟 Gmail 收到的系統通知信。
2.  **務必點擊「回覆 (Reply)」按鈕**。
3.  Gmail 會自動將收件人替換為該名學生的 Email。
4.  **優點**：以此方式回覆，信件標題與引言會包含原始的 AI 對話紀錄連結，方便學生與管理員追溯問題脈絡。請避免複製 Email 另開新信，這會導致脈絡中斷。

## 4. 學生使用指南

### 4.1 公告瀏覽與篩選

![image](https://hackmd.io/_uploads/BkjTdCj8Ze.png)
首頁提供視覺化的公告列表，助您快速掌握資訊：

*   **狀態燈號**：
    *   🟢 **綠色標註**：目前正在申請期限內，可立即申請。
    *   🔴 **紅色標註**：尚未開始或已截止。
*   **前瞻預覽**：在「未逾期」分類中，您可以提前看到即將開放的獎學金，方便提早準備證明文件。
*   **下載功能**：點擊公告卡片或內頁的下載按鈕，系統會自動生成該公告的 PDF 檔案供您保存。

### 4.2 註冊使用者專屬服務

建議您使用學校或個人常用信箱註冊帳號，以解鎖完整功能：

1.  **公告寄送至個人信箱**：
    *   在瀏覽單篇公告時，點擊右下角的「傳送至我的 mail」按鈕，系統即會將該篇公告的完整內容寄送至您的註冊信箱，方便您日後查閱或轉寄。
2.  **接收重要通知**：
    *   當生輔組有重要的新獎助學金發布時，您會直接在信箱收到通知，不錯過任何機會。
3.  **AI 獎學金助理 (AI Assistant)**：
    *   **核心技術**：採用 Google Gemini 系列大型語言模型。
    *   **雙軌檢索**：AI 會優先搜尋「平台內部公告」，若無相關資訊，則會自動「搜尋外部網路」資源。
    *   **智慧標示**：AI 回覆會自動以螢光筆標示重點（如金額、截止日、資格）。
    *   **對話記憶**：系統會保存您的歷史對話。
    *   **真人支援**：若 AI 無法解決您的問題，點擊對話框下方的「尋求承辦人員協助」，系統會將您的問題與 AI 對話紀錄一併寄給生輔組承辦人員。
    >[!Warning]**免責聲明與查證義務**
    >AI 回覆內容僅供參考。系統會附上資料來源連結（內部公告或外部網站），**申請前請務必點擊連結查閱原始文件**，一切申請辦法皆依獎學金官方公告為準。
4.  **個人資料管理**：
    *   您可以隨時更新您的姓名與學號資訊。
    *   系統提供密碼強度檢測功能，協助您設定安全的密碼。
